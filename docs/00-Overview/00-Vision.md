# Vision

## Purpose
Build FlowPilot as a production-grade Instagram automation SaaS using official Meta APIs.

## Goals
- Clean, maintainable architecture
- Security by default
- Free-tier friendly (Render + Supabase)
- Horizontal scalability
- Testability
- Documentation-first development

## Non-Goals
- Browser automation
- Scraping
- Unofficial Instagram APIs

## Success Criteria
- One-command local setup
- Zero architecture rewrites
- Consistent feature implementation
